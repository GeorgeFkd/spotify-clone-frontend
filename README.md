# Spotify Clone

This project was made to practice react and use popular libraries(redux,react-router,material ui etc.)

## Setup
The project is setup using CRA(create-react-app)

## Structure
- Uses vanilla css with material ui for styling
- Project Structure is /components, /redux for reducers and slices, /data for future connection with a backend

## Install And Run
* ```git clone https://github.com/GeorgeFkd/spotify-clone-frontend.git```
* run in a terminal in the same folder:
  * ```npm install``` (installing dependencies)
  * ```npm start``` (start the development server)

> Developed in node.js version 16. (For multiple node versions use: https://github.com/coreybutler/nvm-windows, https://github.com/nvm-sh/nvm as in Node v18 it will not work.)
